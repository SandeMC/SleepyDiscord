# SleepyDiscord
Simple CSS to automatically dim non-hovered elements to prevent OLED burn-in

## Demo

https://github.com/user-attachments/assets/275187fb-b8b3-4a56-8aec-0ed563b32a32

## Installation

Either download and drop sleepyDiscord.theme.css file into your custom client's themes folder manually, or download it via this link if your client supports that: `https://raw.githubusercontent.com/SandeMC/SleepyDiscord/main/sleepyDiscord.theme.css`
